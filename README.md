## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
This application is a gallery application. Shows developers how to use the Gesture Recognizer class. The application has a Label and an ImageView. There are landscape pictures in ImageView, and the names of these pictures in Label. First, we activated the isUserInteractionEnabled property so that the images can interact with the user. Then we added the feature to touch the image using the UITapGestureRecognizer class. Finally, we added this touch feature to the ImageView with the addGestureRecognizer method. We also generated random numbers with arc4random and made the images appear mixed to the user.
